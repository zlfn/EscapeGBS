# 탈출 경기북 (탈북)
![image](https://github.com/zlfn/EscapeGBS/assets/33383685/dd32d5a3-96d0-4b51-bdd1-241ebea3f32e)

2021년에 만들었던 웹게임입니다. 치명적인 버그들의 수정이 반영되지 않아 정상적인 플레이는 불가능합니다.
반응형 웹으로 만들었으며, Rocket 웹 프레임워크를 이용했습니다.

`cargo run`으로 실행
